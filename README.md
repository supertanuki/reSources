# reSources

Through human actions, the planet has been turned into a vast desert. Years later, communities are trying to establish themselves by managing resources more effectively and respecting planetary boundaries. Create your community, be patient, regenerate wood resources, and pray for rain.

- Game concept and development : Richard (supertanuki)
- Sound effects : Nivadra
- Music : Middle Eastern Moon by Abydos_Music
- Assets : Desert map tileset by BeyonderBoy

Game made for [Regenerate 2026 game jam](https://itch.io/jam/regenerate-game-jam-2026/rate/4406421)
